## Design & UI Consistency Instructions

Follow the existing design system of the IDM Smart Tech website without introducing a new visual style.

Website:
https://idmwebsite.vercel.app/

### Theme Consistency
* Use the existing website color palette.
* Use the same typography, font family, font weights, and font sizes already implemented across the website.
* Maintain the current heading hierarchy (H1, H2, H3, H4).
* Match the existing spacing, padding, margins, and section widths.
* Keep the same border radius, shadows, and card styles.
* Reuse the existing button styles, hover effects, and animations.
* Follow the current icon style and icon sizing.
* Use the same background colors, gradients, and accent colors already present on the website.
* Do not introduce any new fonts, colors, or UI components that are inconsistent with the current design.

### Content Layout
Each service page should follow the exact layout and visual structure of the existing website:
* Hero Section
* Trust Section (if applicable)
* About Section
* Service Features
* Benefits
* Why Choose Us
* Process
* Industries We Serve
* Results
* Case Studies
* Testimonials (if applicable)
* FAQs
* Final Call-to-Action
* Related Blogs

### Visual Consistency
* Preserve the same premium, modern, and professional look.
* Use the existing card components and section spacing.
* Maintain responsive behavior across desktop, tablet, and mobile.
* Reuse existing animations and transitions instead of creating new ones.
* Keep the same button styles, hover interactions, and icon treatments throughout the page.

### Development Requirement
Before generating the page, analyze the existing website and reuse its design language, including typography, colors, spacing, cards, buttons, and animation patterns, so the new service page appears as a seamless extension of the current IDM Smart Tech website rather than a newly designed page.
